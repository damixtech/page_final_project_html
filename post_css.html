CSS evoluciona con funciones nativas para temas oscuros

Con la llegada de nuevas funcionalidades como prefers-color-scheme y :has(), CSS está dando pasos importantes hacia un diseño web más inteligente. Estas funciones permiten crear estilos adaptables al sistema del usuario sin necesidad de JavaScript, mejorando tanto el rendimiento como la experiencia de usuario.
